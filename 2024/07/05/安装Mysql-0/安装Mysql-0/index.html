安装Mysql

卸载已安装数据库

安装Mysql8.x版本 yum库

Mysql更新密钥，如若密钥过期，则改为最新的

如果因为一些原因一直无法导入，可直接创建一个文件，本地进行导入（网址看更新密钥的网址）

下面提供2023年的Mysql密钥

使用yum下载安装MySQL8.0

配置MySQL

修改root的权限和密码

用root进入MySQL

将root密码置空

退出MySQL登录，将跳过权限验证选项skip-grant-tables去掉

重启mysqld

重新登录mysql

将MySQL密码改为123456
